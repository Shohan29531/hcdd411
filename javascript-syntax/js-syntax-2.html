  • Access elements by index (starting at 0)

ES6 - let & const

Modern Variable Declaration

  • const - Must be initialized, cannot be reassigned
  • let - Can be reassigned, block-scoped
  • Prefer const by default, use let when needed
⚠️ let/const vs var:
  • let/const are block-scoped, var is not
  • var can be used before declaration (hoisting)
  • var allows duplicate names, let/const do not
