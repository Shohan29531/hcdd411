JavaScript Variables

Declaring Variables

  • Use var keyword (older style)
  • Use let keyword (modern, recommended)
  • Use const for constants — cannot be reassigned
  • No need to declare a data type!
⚠️ Variable Naming Rules:
  • Case sensitive: myVar ≠ myvar
  • Can contain: letters, numbers, $, _
  • Cannot start with a number
  • Cannot use reserved keywords (like if, for, etc.)

Array Method: filter()

Creates a new array with only the elements that pass a test

How it works:
filter() runs a function on each item. If the function returns true, the item is kept. If false, it's removed. The original array is never modified.
  • Takes a callback function that returns true or false
  • Returns a new array — the original is unchanged
  • Perfect for finding items that match a condition
