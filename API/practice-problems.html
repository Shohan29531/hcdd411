Practice 1 — JavaScript: Sum, Average, Max

Task:
  • Given an array of numbers, compute sum, avg, and max.

  • Make average show 2 decimals (example: 10.25).
Hint (Idea):
  • Loop through the array, add into sum, and update max when you see a bigger number.
Hint (Syntax):
  • for (let n of nums) { ... }
  • avg = sum / nums.length
